# Dolly's Diary - デザイン計画書

## コンセプト

「カレンダー＆日記」アプリ。温かみのあるクリーム系の配色で、毎日の出来事を手軽に記録できるシンプルなダイアリーアプリ。ウィジェット機能は含まない。

## スクリーン一覧

### 1. カレンダー画面（ホーム）
- 月間カレンダーを表示
- 日記が記録されている日にはドットインジケーターを表示
- 日付をタップすると、その日の日記詳細画面へ遷移
- 月の切り替え（前月・次月）ボタン
- 今日の日付がハイライト表示

### 2. 日記一覧画面
- 日記エントリーをカード形式で新しい順に一覧表示
- 各カードには日付、タイトル、本文の冒頭、気分アイコンを表示
- 検索バーで日記を検索可能

### 3. 日記詳細画面
- 選択した日の日記を全文表示
- 日付、タイトル、気分、本文を表示
- 編集ボタン、削除ボタン

### 4. 日記作成・編集画面
- タイトル入力フィールド
- 気分選択（絵文字ベース：😊😢😡😴🥰など5種類）
- 本文入力（複数行テキストエリア）
- 保存ボタン

### 5. 設定画面
- テーマ切り替え（ライト/ダーク）
- アプリ情報

## 主要コンテンツと機能

| 画面 | コンテンツ | 機能 |
|------|-----------|------|
| カレンダー | 月間カレンダーグリッド、日記インジケーター | 日付選択、月切替、日記作成への遷移 |
| 日記一覧 | 日記カードリスト | 一覧表示、検索、日記詳細への遷移 |
| 日記詳細 | 日記全文、気分、日付 | 閲覧、編集、削除 |
| 日記作成/編集 | 入力フォーム | タイトル・気分・本文の入力と保存 |
| 設定 | 設定項目リスト | テーマ切替 |

## キーユーザーフロー

### 日記を書く
1. カレンダー画面で日付をタップ
2. 日記がない場合 → 日記作成画面が開く
3. タイトル、気分、本文を入力
4. 保存ボタンをタップ → カレンダーに戻り、その日にドットが表示される

### 日記を読む
1. カレンダー画面で日記のある日付をタップ → 日記詳細画面
2. または、日記一覧タブから目的の日記カードをタップ → 日記詳細画面

### 日記を編集する
1. 日記詳細画面で編集ボタンをタップ
2. 日記編集画面で内容を修正
3. 保存ボタンをタップ → 日記詳細画面に戻る

### 日記を削除する
1. 日記詳細画面で削除ボタンをタップ
2. 確認ダイアログ → 削除確定 → カレンダー画面に戻る

## タブ構成

| タブ | アイコン | 説明 |
|------|---------|------|
| カレンダー | calendar-today | メインのカレンダー画面 |
| 日記一覧 | menu-book | 日記の一覧表示 |
| 設定 | settings | アプリ設定 |

## ナビゲーション構造

```
(tabs)/
  ├── index.tsx          → カレンダー画面（ホーム）
  ├── entries.tsx         → 日記一覧画面
  └── settings.tsx        → 設定画面

diary/
  ├── [date].tsx          → 日記詳細画面
  └── edit.tsx            → 日記作成・編集画面
```

## 注意事項

- ウィジェット機能は含まない（`react-native-android-widget` は使用しない）
- データはローカル（AsyncStorage）に保存
- 広告機能は含まない
- 認証機能は含まない
